# Лабораторная работа  
## Метод Гаусса с выбором главного элемента по строке

### Описание

Программа решает систему линейных уравнений

\[
Ax=b
\]

методом Гаусса с выбором главного элемента по строке.

В программе реализована функция `gaussMethod(A, b)`, которая:
- принимает матрицу `A` и вектор правой части `b`;
- проверяет корректность размеров входных данных;
- строит разложение
  \[
  AQ = LU,
  \]
  где `Q` — матрица перестановки столбцов, `L` — нижнетреугольная матрица, `U` — верхнетреугольная матрица;
- решает системы
  \[
  Lz=b
  \quad \text{и} \quad
  Uy=z
  \]
  методами прямой и обратной подстановки;
- восстанавливает решение исходной системы по формуле
  \[
  x = Qy;
  \]
- возвращает `L`, `U`, `Q`, `x`.

Дополнительно программа:
* запускает функцию на нескольких тестах;
* сравнивает найденное решение с `numpy.linalg.solve`;
* выводит нормы
  \[
  \|AQ-LU\|,\quad \|x-x_{lib}\|,\quad \|Ax-b\|.
  \]

## Вывод

Программа была протестирована на трёх примерах. Во всех тестах разложение
\[
AQ = LU
\]
построено корректно, потому что
\[
\|AQ - LU\|
\]
во всех случаях почти равна нулю.

Найденные решения совпадают с результатами функции `numpy.linalg.solve`. Норма разности
\[
\|x - x_{lib}\|
\]
имеет порядок \(10^{-16}\), то есть тоже относительно мала.

Норма невязки
\[
\|Ax - b\|
\]
также близка к нулю во всех тестах, следовательно найденные решения удовлетворяют исходным системам.

По результатам тестирования можно сделать вывод, что реализованный метод Гаусса с выбором главного элемента по строке работает. Программа корректно строит матрицы \(L\), \(U\), \(Q\), находит решение системы и даёт правильный результат, совпадающий с библиотечным решением.
